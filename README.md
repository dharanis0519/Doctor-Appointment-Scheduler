# Doctor-Appointment-Scheduler

##  Project Overview
This is a responsive Appointment Scheduling web application developed using **HTML, CSS, and JavaScript**.  
The application allows users to book Appointment, edit, delete, and manage appointments with a calendar view.

##  Features

-  Calendar view for appointments  
-  Book new appointments  
-  Edit existing appointments  
-  Delete appointments  
-  Search by patient name and doctor  
-  Filter by date / date range  
-  Fully responsive (Mobile, Tablet, Desktop)  
-  Data stored using local Storage
